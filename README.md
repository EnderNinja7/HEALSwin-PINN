# HEALSwin-PINN

# Dataset
Model-1 from DeepLenseSim [GitHub](https://github.com/mwt5345/DeepLenseSim/tree/main)

## Models
Base Swin Transformer
Swin Transformer with Relativistic Physics Encoder
HEAL-Swin
HEAL-Swin with Relativistic Physics Encoder
